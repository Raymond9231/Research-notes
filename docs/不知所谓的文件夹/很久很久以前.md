# 很久很久以前

我开始编辑这个网站的第一批页面

## 网站简介

[取决于读者认为我是谁]，现在正在度过自己的又一个假期。依赖AI完成科研，也大方承认过相较于几年前程序报错拼尽全力也无法在stackoverflow找到完全匹配的解决方案，AI的出现让自己终于有能力在学校快速推进一个完全没接触过领域的科研项目，也让自己无论多忙也不至于在组会上一个字也憋不出。

同时，此人也对于这种跨度过大的科技进步十分担忧。偶然依靠AI完成了以前认为艰难的任务会让人狂喜，但是每次艰难的任务都是由AI完成只会让人感觉自己真的很菜。更让人恐惧的是，即便认识到了这一点，本人依然义无反顾地跳入深渊，放任AI替代原生大脑思考。

因此，在我尚还能思考的时候，我决定把我对自己正在做的一些项目的思考进程和相关文献的阅读理解都记下来。我并不奢望一夜之间就能开始细细品味AI的思考过程并从中学到大量的知识，我只希望与人交流我的专业时的时候我能充满热情地描绘出我所创造的世界的一分一毫。
